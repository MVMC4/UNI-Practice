# ☕ Java OOP Notes

A structured collection of notes and practice code covering core **Object-Oriented Programming** concepts in Java.

---

## Overview

This repository serves as a personal reference for Java OOP fundamentals studied during university. Each folder focuses on a specific concept and contains markdown notes, code examples, or both.

| Folder | Type | Description |
|---|---|---|
| `Constructors/` | Notes + Code | How constructors work, including extended/inherited constructors |
| `Method/` | Notes | Method definitions, signatures, and usage |
| `Modifiers/` | Notes | Access and non-access modifiers (`public`, `static`, `final`, etc.) |
| `Objects/` | Notes + Code | Object creation, attributes, and key observations |
| `This/` | Notes | The `this` keyword and its use cases |
